# PixelLab.ai Prompts — Clowder & Crest

Pixel art assets for a medieval cat guild management game. The setting is a late-medieval European town (think 1300s–1400s) — stone buildings, candlelight, rain, cobblestones. The tone is cozy but slightly dark, like a Redwall or medieval tapestry aesthetic.

## Style Guide

- **Resolution:** 32x32 for sprites/items, 64x64 for portraits, 320x180 for scene backgrounds
- **Palette:** Warm muted earth tones — browns (#8b7355, #6b5b3e), dark stone (#2a2520, #3a3530), warm amber (#c4956a, #dda055), blue-grey (#6b8ea6), deep green (#4a6a4a). Black backgrounds (#1c1b19).
- **Style:** Low-detail chunky pixel art, 2–3 color shading per object, no outlines or thin black outlines only. Think SNES/GBA RPG item icons.
- **Perspective:** 3/4 top-down isometric for room interiors, side-view for cat portraits, flat for UI icons.

## File Naming Convention

Save each asset with this naming pattern:
```
portraits/  wildcat.png, russian_blue.png, tuxedo.png, maine_coon.png, siamese.png
sprites/    wildcat_idle_sw.png, wildcat_idle_se.png, wildcat_idle_nw.png, wildcat_idle_ne.png,
            wildcat_walk_sw.png, wildcat_walk_se.png, wildcat_walk_nw.png, wildcat_walk_ne.png,
            wildcat_sleep.png, wildcat_work.png (same pattern for each breed)
furniture/  straw_bed.png, woolen_blanket.png, cushioned_basket.png (etc.)
tiles/      stone_floor_dark.png, stone_floor_light.png, wood_plank.png
scenes/     town_square.png, guildhall_exterior.png, sleeping_quarters.png
ui/         fish.png, paw.png, scroll.png, sword.png, heart.png, star.png, home.png, shield.png, mouse.png, moon.png
puzzles/    flour_sack.png, grain_crate.png, pew.png, candelabra.png, market_stall.png, cart.png, cat_block.png
```

Place generated files in `/var/www/clowder.stephens.page/public/assets/` under the corresponding subfolder.
