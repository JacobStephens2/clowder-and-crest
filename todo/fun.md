# Clowder & Crest — Fun Analysis

Based on the frameworks in "What Makes Games Fun?" (Koster, Csikszentmihalyi, SDT, MDA)

## Where the Game Succeeds

### Pattern Mastery (Koster)
- **7 minigame types** each present distinct learnable patterns — slide blocks (spatial), sokoban (planning), chase (pathfinding), fishing (timing), hunt (reflexes), brawl (combat positioning), nonogram (logic)
- **Job stat matching** rewards learning which cat fits which job
- **Economy** has patterns to master (upkeep scaling, combo chains, merchant timing)
- **Reputation system** creates learnable cause-and-effect (shadow pays more but costs bonds)

### Flow State (Csikszentmihalyi)
- **3 difficulty tiers** per minigame scale challenge with progression
- **Cat stats affect minigame parameters** (hunting adds time, etc.) — natural difficulty curve as cats level up
- **Chapter-gated content** prevents overwhelming new players

### Self-Determination Theory
- **Autonomy**: Player chooses which cat takes which job, which minigame approach, which reputation path, how to furnish rooms, which cats to recruit
- **Competence**: XP and leveling, specialization at level 5, star ratings on puzzles, combo streaks
- **Relatedness**: Bond conversations between cats create emotional investment; the guild is a found family

### MDA 8 Kinds of Fun
| Type | Present? | How |
|---|---|---|
| Sensation | Yes | Pixel art sprites, music tracks, sound effects, rain ambience |
| Fantasy | Yes | You ARE a cat running a medieval guild |
| Narrative | Strong | 7-chapter story arc, plague siege, inquisition, rival guild, 48 bond conversations |
| Challenge | Strong | 7 minigame types with difficulty scaling |
| Fellowship | Partial | Bond system between cats (not multiplayer) |
| Discovery | Yes | Chapter unlocks, new breeds, merchant items, building exploration in town map |
| Expression | Yes | Name your cat, choose specialization, decorate rooms, reputation path |
| Submission | Yes | Cozy tone, 3-minute days, stationed passive income |

## Where the Game Could Improve

### Fun Killers Present
1. **Repetition without novelty** — After mastering all minigames, the core loop (take job, minigame, collect fish) can feel samey. The 7 minigame types help but each one is finite in variety.
2. **Weak feedback on some actions** — Economy changes (fish earned/lost) could use more visual celebration.
3. **Choices that don't matter enough** — Some job choices feel interchangeable when you have strong cats. The specialization system helps but kicks in late (level 5).
4. **Progress plateau in mid-game** — Between chapters 3-5, progression can feel slow if the player has already recruited most cats.

### Improvements to Make
1. **More celebration on victories** — Add screen shake and particle effects when completing a minigame with 3 stars
2. **Streak/combo visual feedback** — When hitting a job combo chain, show a visible multiplier badge
3. **Mid-game events** — More random events between chapters to break routine
4. **Fish earning celebration** — Animate fish count in status bar when earned
